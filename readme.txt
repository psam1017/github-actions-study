Inflearn 에서 박재성 님의 강의 '비전공자도 이해할 수 있는 CI/CD 입문·실전' 를 수강하면서 실습 및 개인화한 프로젝트입니다.

Github Actions 란 CI/CD 를 돕는 도구로서 Github Repository 에서 발생한 이벤트를 감지하여 각종 명령어와 플러그인을 실행할 수 있습니다.

Workflow 기능을 사용하기 위해서는 프로젝트 최상단에서부터 .github/workflows 디렉토리를 생성하고 그 안에 yml 파일을 통해 실행시킬 수 있습니다.

AWS CodeDeploy 를 사용하려면 appspec.yml 을 생성해야 합니다.

각 파일의 주석 및 step name 을 통해 해설을 달았으며, 노션으로 정리한 페이지는 공개하지 않습니다.

학습이 필요하신 분은 언급된 강의를 수강하시기를 추천드립니다.
